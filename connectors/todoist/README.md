# Todoist Connector for Plot

Sync your Todoist projects and tasks with Plot.

## Features

- OAuth authentication with Todoist
- Sync tasks from Todoist projects to Plot
- Real-time updates via webhooks
- Comments synced as notes (with history backfill on first sync)
- Subtasks synced as notes with todo tags
- Per-project sections surfaced as statuses
- Create Todoist tasks from Plot
- Two-way sync of title, assignee, section, completion, comments, and edits
- File attachments on comments (inbound and outbound)
